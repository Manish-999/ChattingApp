Welcome to our Chat Application. This chat application is developed for learning purpose, here we use Angular as Front End, express as Server and MongoDB as DataBase

These are the steps for use this Application. So for use it follow all these step by step :-

  1. Open this Application.
  2. In this Application you provide two inputs. Input are Name, Room Id.
  3. Now Enter your Name and unique Room Id.
  4. All those Name who have same Room, they connected in Chat Application.
  5. Share your Room Id to your Friends to connect.
  6. Now at Bottom press the button to start this Application and follow these steps to start Chatting.
